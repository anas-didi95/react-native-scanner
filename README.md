## Setup
Clone this repo to your desktop.

Run the command to install dependencies.
```
$ yarn install
```

Run the development server.
```
$ yarn start
```

---

## Features
* Scan QR code
* Display decoded content
* Copy decoded content
* Open link if decoded content is URL
